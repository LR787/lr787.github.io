Markdown 笔记3之 《字体》

加粗

要加粗的文字左右分别用两个*号包起来

**font**

实际效果：

font

这里是加粗的字体哦

斜体

要倾斜的文字*左右分别用一个号包起来**

*font*

实际效果：

font

这里是斜体效果哦

删除线

要加删除线的文字左右分别用两个 ~~ 号包起来
实际效果：

~~deletion~~

这里是被删除线划掉的文字哦

小总结： 加粗用一对两个，斜体用一对，中间把文字包起来就可以了

Markdown 笔记5《 超链接》

超链接

语法：

[超链接名](超链接地址 "超链接title")

title可加可不加
示例：

[简书](http://jianshu.com)
[百度](http://baidu.com)
实际效果：

简书
百度

注：Markdown本身语法不支持链接在新页面中打开，貌似简书做了处理，是可以的。别的平台可能就不行了，如果想要在新页面中打开的话可以用html语言的a标签代替。

<a href="超链接地址" target="_blank">超链接名</a>

示例<a href="https://www.jianshu.com/u/1f5ac0cf6a8b" target="_blank">简书</a>

Markdown 笔记6《列表（无序+有序）》

无序列表

语法：

无序列表用 - + * 任何一种都可以

- 列表内容
+ 列表内容
* 列表内容

注意：*- + * 跟内容之间都要有一个空格*

实际效果：

  • 列表内容
  • 列表内容
  • 列表内容

有序列表

语法：

数字加点

1.列表内容
2.列表内容
3.列表内容

注意：序号跟内容之间要有空格

效果如下：
1.列表内容
2.列表内容
3.列表内容

Markdown笔记7 《代码块》

Markdown代码块

对于程序员来说这个功能是必不可少的，插入程序代码的方式有两种，一种是利用缩进(tab), 另一种是利用英文“ ` ”符号（一般在ESC键下方，和~同一个键）包裹代码。

（1）代码块：缩进 4 个空格或是 1 个制表符。

（2）行内式：如果在一个行内需要引用代码，只要用反引号 引起来就好（一般在ESC键下方，和~同一个键）

`code`

效果： code

（3）多行代码块与语法高亮：在需要高亮的代码块的前一行及后一行使用三个单反引号“ ` ”包裹，就可以了。
